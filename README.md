# 设计思想  
1、 模拟性能测试工具JMeter的思路，实现接口测试与性能测试的Web化；  
2、 计划有产品模块、接口测试用例模块、性能测试用例模块、任务队列模块、测试报告模块等；  
3、 前期采用Django+Bootstrap前后端不分离的模式实现功能，后期再严格按照RESTful的风格来编程。
